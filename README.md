# El País Más Rapero

Cartografía visual del rap chileno: artistas, discos, ciudades, colaboraciones, eras y evidencia curatorial.

Proyecto personal de Seo2 para mapear el rap chileno como una red viva y ampliable.

## Stack

- Next.js App Router
- React
- TypeScript
- D3 para el grafo interactivo
- Tests con `tsx --test`

## Desarrollo

```bash
npm install
npm run dev
```

## Validación

```bash
npm test
npm run typecheck
npm run lint
npm run build
```

## Criterio editorial

Spotify puede ayudar como fuente de catálogo musical, pero no define la historia. La capa curatorial del proyecto combina fuentes externas, revisión manual y decisiones editoriales explícitas.
