# Opciones de nombre

## Favoritos fuertes

1. **Mapa del Flow Chileno**  
   Claro, memorable, comunica mapa + rap/flow. Puede crecer a urbano después.

2. **Archivo del Rap Chileno**  
   Más serio y patrimonial. Ideal si queremos tono documental/cultural.

3. **Rapografía Chilena**  
   Nombre propio, mezcla rap + cartografía. Diferente y fácil de recordar.

4. **La Red del Rap Chileno**  
   Directo para un sitio centrado en grafos y colaboraciones.

5. **Flowgrafía**  
   Corto, visual, moderno. Podría servir como marca general.

## Mi recomendación

Para este proyecto, mis 3 mejores son:

1. **El País Más Rapero** — elegido como nombre principal. Tiene personalidad, orgullo local y deja espacio para mapear escenas, eras y territorios del rap chileno.
2. **Rapografía Chilena** — el más distintivo alternativo.
3. **Atlas del Rap Chileno** — el más claro/patrimonial.
4. **Mapa del Flow Chileno** — el más accesible y moderno.

Nombre principal elegido para la marca pública: **El País Más Rapero**. Nombre técnico usado en carpeta/paquete por ahora: **rap-chileno-grafo**.
